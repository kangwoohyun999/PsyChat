# ConditionChatApp (Expo)

이 프로젝트는 Expo 기반 React Native 앱입니다.
주요 기능:
- 날짜 선택 가능한 채팅으로 컨디션 기록
- 키워드(유사어) 추출 → 가중치 기반 감성 판정
- 누적 상태 파이차트 및 단어별 빈도 그래프
- 날짜별 히스토리 조회

## 실행 방법

1. Node.js 및 Expo CLI 설치
2. 의존성 설치:
   npm install
3. Expo 시작:
   npx expo start
